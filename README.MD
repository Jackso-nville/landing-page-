# Agency Landing Page

A responsive single-page landing website for a digital agency offering
Social Media Management, SEO, Design, and Advertising services.

Built using Vanilla HTML, CSS, and JavaScript with Vite.

---

## Sections

- Hero
- About Us
- Services
- What We Do
- Testimonials
- Collaborate With Us
- Footer

---

## Tech Stack

- HTML5
- CSS3 (Vanilla, modular structure)
- JavaScript (ES Modules)
- Vite

---

## Project Structure

The project is organized as follows:

```
index.html          # Main HTML file
package.json        # Project metadata and dependencies
README.MD          # Project documentation
assets/            # Folder for assets
    images/        # Images used in the project
public/            # Publicly accessible files
    images/        # Images for public use
src/               # Source files
    main.js       # Main JavaScript file
    scripts/      # JavaScript files for specific functionalities
        form.js   # Form handling scripts
        navigation.js # Navigation scripts
    styles/       # CSS files for styling
        base.css  # Base styles
        components.css # Component-specific styles
        layout.css # Layout styles
        main.css   # Main styles
        variables.css # CSS variables
```

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd agency-creative
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Usage

Open your browser and navigate to `http://localhost:3000` to view the project.
